novastar.js

This JavaScript library (written in TypeScript) is an attempt, using reverse engineering, to write an API wrapper for working with NovaStar devices. Compatible with Windows, Linux and macOS.

This monorepo consists of the following packages:

  • @novastar/codec - core API for communication with devices using NovaStar protocol
  • @novastar/serial - binding to a serial interface
  • @novastar/net - binding to a network interface
  • @novastar/native - structures and APIs in typescript automatically generated from decompiled NovaStar .NET libraries
  • @novastar/screen - API wrapper
